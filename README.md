## рҹҺҜ What is HumanOS?

**HumanOS** is a next-generation personality assessment platform that combines professional psychological assessments with stunning visual experiences.

> *"Discover yourself through science and art."*

### Key Philosophy

| Principle | Description |
|-----------|-------------|
| рҹ”’ **Privacy First** | All data stays in your browser, never uploaded |
| рҹ§  **Scientific** | Based on established psychological research |
| рҹҺЁ **Beautiful** | Modern glassmorphism design with smooth animations |
| вҡЎ **Fast** | Built with Vite for lightning-fast performance |

## рҹҡҖ Quick Start

### Requirements

- Node.js >= 18.0.0
- npm >= 9.0.0 or pnpm >= 8.0.0

### Installation

```bash
# Clone repository
git clone https://github.com/badhope/HumanOS.git

# Navigate to directory
cd HumanOS

# Install dependencies
npm install
# or
pnpm install
```

### Development

```bash
# Start dev server
npm run dev

# Open http://localhost:5173
```

### Build

```bash
# Production build
npm run build

# Preview build
npm run preview
```

### Code Quality

```bash
# Run ESLint
npm run lint

# TypeScript check
npm run typecheck
```
