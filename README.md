## 🎯 什么是心镜？

心镜是一款**专业心理测评与成长训练平台**，就像 Keep 帮你塑造身体一样，心镜帮你探索内心、提升心理素养。

> 💡 **核心理念**: 测评不是终点，成长才是目的

### ✨ 三大特色

| 🧩 **游戏化测评** | 🏋️ **训练式成长** | 📊 **数据化反馈** |
|:---:|:---:|:---:|
| 43+ 专业量表 | 8 大成长赛道 | 可视化进度追踪 |
| ACG 沉浸式主题 | 41 套专项训练 | 智能推荐系统 |
| 科学理论基础 | 4 级进阶体系 | 个人成长档案 |

### 本地运行
```bash
# 克隆仓库
git clone https://github.com/badhope/MindMirror.git
cd MindMirror

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```
