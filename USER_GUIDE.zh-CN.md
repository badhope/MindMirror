# MindMirror 用户指南

> 📖 Other languages: [English](USER_GUIDE.md)

一份给普通用户的、不讲术语的 MindMirror 使用说明。如果只是想试试,
直接打开 **[mindmirror.app](https://badhope.github.io/MindMirror/)**,有不清楚的再回来看。

---

## 1. 这个应用能做什么

| 功能                                | 你能拿到什么                                                    |
| ----------------------------------- | --------------------------------------------------------------- |
| **性格测评**(大五人格, 60 题)        | 测出你在五个性格大维度上的位置                                  |
| **压力测评**(PSS-10)                | 快速判断"最近是不是太累了"                                      |
| **焦虑测评**(GAD-7)                 | 心理科医生常用的初筛工具,7 道题                                |
| **社会支持**(SSRS, 43 题)            | 评估你身边的关系网: 主观感受、客观资源、利用程度              |
| **职业倦怠**(MBI-GS, 40 题)          | 工作中的情感耗竭、犬儒主义、效能感; 附 4 种倦怠原型           |
| **生活满意度**(SWLS, 40 题)          | 关系 / 健康 / 成就 / 成长 / 意义 / 日常六主题的全貌             |
| **心理韧性**(CD-RISC-10, 40 题)      | 适应性 / 关系支持 / 意义 / 自我效能 / 乐观五大子维度           |
| **每日心情**                        | 每天一句话打卡,几周后自动出趋势图                              |
| **训练计划**                        | 短小的 CBT 风格计划,自己安排时间做                             |
| **历史记录**                        | 所有做过的测评,放在一个时间轴上对比                            |

## 2. 四步走

1. **挑一个量表。** 首页能看到所有可用的测评,每个都写了"几分钟做完、测什么"。
2. **凭第一反应答。** 没有对错,真实就好,越真实越有用。
3. **看报告。** 一段简短总结 + 一张雷达图 + 每个子维度的解读。
4. **过段时间再来。** 结果会存到你的"历史"里。压力和焦虑建议每 4-6 周复测一次 —— 看趋势,不要看单次。

## 3. 两种使用方式

### 3a. 在线 demo(零门槛)

打开 **[badhope.github.io/MindMirror](https://badhope.github.io/MindMirror/)**,所有数据存在你浏览器的
`localStorage` 里。我们看不到。换电脑、换浏览器就从零开始。

好处:不用注册,不用安装。
缺点:数据不跨设备、不跨浏览器。

### 3b. 自部署(你或你公司自己的服务器)

这份文档对应 `main` 分支的纯静态版,没有后端。
如果你想要多人自托管版本(FastAPI + Postgres、OAuth、自己挂反向代理),
请在同一个仓库切到 `server` 分支,按那边的 README 走。

## 4. 隐私,用大白话讲

- 我们**不接广告**。没有"赞助内容"。
- 我们**不接第三方统计**。没有 Google Analytics,没有 Facebook 像素,什么都没有。
- 在线 demo 模式下,数据只在你浏览器里。关掉页面,清掉浏览数据,就真的没了。
- 自部署模式下,数据只在你的数据库里。我们没有远程访问通道。
- 因为我们**根本没有你的数据**,所以不存在"卖"或者"分享"。

如果这条说明里有没回答到的疑问,在 [github.com/badhope/mindmirror/issues](https://github.com/badhope/mindmirror/issues) 提 issue(目前没有公开的支持邮箱;在 issue 页面里选"🔒 private contact"即可不公开)。

## 5. 常见问题

**Q. 这是诊断吗?**
A. 不是。我们用的是临床医生**起点判断**用的量表,不是最终诊断。如果你有持续的困扰,请联系专业心理咨询师或医院。

**Q. 我能给团队用吗?**
A. 可以。自部署版本让团队所有人共用一个数据库,数据都在你公司手里。HR 维度的汇总报告正在排期。

**Q. 多长时间做一次?**
A. 性格比较稳定,一年一次足够;压力和焦虑会随生活变化,4-6 周一次比较好,看趋势,不要看单次。

**Q. 我手滑删了一条记录,能恢复吗?**
A. 在线 demo:**不能**。`localStorage` 删了就是删了。
自部署:**可以**。从你的 Postgres 备份恢复就行。

**Q. 有手机 app 吗?**
A. 网页版支持 PWA,你可以"加到主屏幕"离线使用。我们暂时没有原生 app 的计划,Web 栈就是产品本身。

## 6. 键盘快捷键

做测评的时候可以用:

| 键                        | 动作            |
| ------------------------- | --------------- |
| `1`–`5`(GAD-7 是 `1`–`7`) | 选第 N 个答案   |
| `←` / `→`                 | 上一题 / 下一题 |
| `Enter`                   | 最后一题时提交  |
