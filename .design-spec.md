# HumanOS 设计规范文档

## 项目定位
**面向企业推广的专业心理测评平台**
- 目标用户：企业用户、研究机构、专业人士
- 核心价值：数据驱动的权威分析，全面精准的用户画像
- 品牌调性：专业、权威、可信、高效

---

## 1. 设计风格

### 核心美学：现代极简主义
- **设计理念**：少即是多，功能至上
- **视觉特征**：
  - 大量留白，提升呼吸感
  - 清晰的视觉层次
  - 简洁的几何形状
  - 精确的对齐和网格系统
  - 克制的装饰元素

### 设计原则
1. **专业性优先**：所有设计决策服务于专业性和可信度
2. **数据驱动**：突出数据可视化，强调分析结果
3. **高效交互**：减少用户认知负担，快速完成任务
4. **响应式设计**：完美适配桌面、平板、移动端

## 3. 字体系统

### 字体家族
```css
/* 主字体 - 现代专业 */
--font-primary: 'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif;

/* 中文字体 - 清晰易读 */

/* 等宽字体 - 数据展示 */
--font-mono: 'JetBrains Mono', 'Fira Code', 'Consolas', monospace;
```

### 字体层级
```css
/* 标题层级 */
--text-h1: 48px;   /* 主标题，用于页面标题 */
--text-h2: 36px;   /* 二级标题，用于区块标题 */
--text-h3: 28px;   /* 三级标题，用于卡片标题 */
--text-h4: 24px;   /* 四级标题，用于小节标题 */
--text-h5: 20px;   /* 五级标题，用于列表标题 */
--text-h6: 18px;   /* 六级标题，用于辅助标题 */

/* 正文层级 */
--text-lg: 18px;   /* 大正文，用于重要内容 */
--text-base: 16px; /* 基础正文，用于常规内容 */
--text-sm: 14px;   /* 小正文，用于辅助信息 */
--text-xs: 12px;   /* 最小正文，用于标签和提示 */

/* 行高 */
--leading-tight: 1.25;   /* 标题行高 */
--leading-normal: 1.5;   /* 正文行高 */
--leading-relaxed: 1.75; /* 宽松行高 */

/* 字重 */
--font-light: 300;     /* 轻字重 */
--font-normal: 400;    /* 常规字重 */
--font-medium: 500;    /* 中等字重 */
--font-semibold: 600;  /* 半粗字重 */
--font-bold: 700;      /* 粗字重 */
```

### 字体应用规则
- **标题**：使用半粗字重（600），紧凑行高（1.25）
- **正文**：使用常规字重（400），标准行高（1.5）
- **强调**：使用中等字重（500）或半粗字重（600）
- **数据**：使用等宽字体，确保数字对齐

## 4. 图标系统

### 图标库
- **主要图标库**：Lucide Icons（现代、清晰、专业）
- **辅助图标库**：Heroicons（补充性图标）

### 图标风格
- **线条粗细**：2px（标准）、1.5px（精细）
- **圆角半径**：2px（现代感）
- **尺寸规范**：
  - 小图标：16px × 16px
  - 标准图标：20px × 20px
  - 大图标：24px × 24px
  - 超大图标：32px × 32px

### 图标应用规则
- **导航图标**：使用标准尺寸（20px），线条粗细2px
- **功能图标**：使用大尺寸（24px），线条粗细2px
- **装饰图标**：使用超大尺寸（32px），线条粗细1.5px
- **颜色**：继承父元素颜色，或使用中性色（neutral-500/600）

## 5. Logo规范

### Logo设计理念
- **核心元素**：人形图标 + 品牌名称
- **设计风格**：现代、简洁、专业
- **颜色方案**：主色（primary-600）+ 深色（neutral-800）

### Logo尺寸规范
```css
/* 桌面端 */
--logo-height-desktop: 40px;

/* 平板端 */
--logo-height-tablet: 36px;

/* 移动端 */
--logo-height-mobile: 32px;

/* 最小尺寸 */
--logo-height-min: 24px;
```

### Logo应用规则
- **位置**：页面左上角（导航栏）
- **间距**：左右各16px，上下居中
- **响应式**：根据屏幕尺寸自动调整
- **点击区域**：Logo可点击返回首页

## 9. 动画系统

### 动画时长
```css
--duration-fast: 150ms;    /* 快速动画 */
--duration-normal: 300ms;  /* 标准动画 */
--duration-slow: 500ms;    /* 慢速动画 */
```

### 缓动函数
```css
--ease-in: cubic-bezier(0.4, 0, 1, 1);
--ease-out: cubic-bezier(0, 0, 0.2, 1);
--ease-in-out: cubic-bezier(0.4, 0, 0.2, 1);
--ease-bounce: cubic-bezier(0.68, -0.55, 0.265, 1.55);
```

### 动画应用规则
- **悬停效果**：使用 duration-fast + ease-out
- **页面过渡**：使用 duration-normal + ease-in-out
- **模态框**：使用 duration-normal + ease-out
- **数据加载**：使用 duration-slow + ease-in-out

## 10. 响应式断点

### 断点规范
```css
/* 移动端优先 */
--breakpoint-sm: 640px;   /* 小屏手机 */
--breakpoint-md: 768px;   /* 平板竖屏 */
--breakpoint-lg: 1024px;  /* 平板横屏 */
--breakpoint-xl: 1280px;  /* 小桌面 */
--breakpoint-2xl: 1536px; /* 大桌面 */
```

### 响应式规则
- **移动端**：单列布局，全宽组件
- **平板端**：双列布局，适当留白
- **桌面端**：多列布局，居中对齐

```css
--chart-1: #3B82F6;  /* 主色 */

--chart-5: #8B5CF6;  /* 紫色 */

```

### 图表应用规则
- **柱状图**：使用主色系，渐变填充
- **折线图**：使用主色，线条粗细2px

- **雷达图**：使用半透明填充，突出数据区域

## 13. 无障碍设计

### 对比度要求
- **正文文字**：对比度 ≥ 4.5:1
- **大标题**：对比度 ≥ 3:1
- **图标**：对比度 ≥ 3:1

### 交互要求
- 所有交互元素支持键盘导航
- 提供清晰的焦点状态
- 使用语义化HTML标签
- 为图片提供alt文本

---

## 14. 品牌元素

### 品牌关键词
- **专业**：权威、可信、数据驱动
- **高效**：快速、流畅、无干扰
- **现代**：简洁、清晰、美观
- **人性化**：友好、易懂、贴心

### 品牌应用
- **文案风格**：专业但不晦涩，简洁但不冷漠
- **视觉风格**：现代极简，突出数据和功能
- **交互风格**：高效流畅，减少用户负担
