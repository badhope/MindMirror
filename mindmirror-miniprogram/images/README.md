# 图片资源目录

请在此目录下添加以下图片文件：

## 需要的图片

### 1. logo.png
- 用途：首页显示的应用 Logo
- 建议尺寸：200x200 像素
- 格式：PNG（支持透明背景）

### 2. 首页图标
- home.png - 未选中状态（建议 81x81px）
- home-active.png - 选中状态（建议 81x81px）

### 3. 测评图标
- assess.png - 未选中状态（建议 81x81px）
- assess-active.png - 选中状态（建议 81x81px）

## 图标设计建议

- 使用简洁的线条图标
- 未选中状态使用灰色 (#8a8a8a)
- 选中状态使用品牌红色 (#e94560)
- 保持图标的视觉一致性

## 临时方案

如果暂时没有图标，可以：
1. 从现有网页应用中提取图标
2. 使用在线图标库（如 iconfont、iconpark 等）
3. 使用简单的纯色占位图
