## 🚀 快速开始

### 🌐 在线版本

👉 **访问**: [https://mindmirror.dpdns.org](https://mindmirror.dpdns.org)

### 💻 本地开发

#### 环境要求

- Node.js >= 18.0.0
- npm >= 9.0.0（或 yarn、pnpm）

#### 安装步骤

```bash
# 1. 克隆仓库
git clone https://github.com/badhope/MindMirror.git
cd MindMirror

# 2. 安装依赖
npm install

# 3. 启动开发服务器
npm run dev

# 4. 打开浏览器
# 访问 http://localhost:5173
```

#### 生产环境构建

```bash
# 构建项目
npm run build

# 预览生产构建
npm run preview
```

## 🎮 如何使用

### 步骤 1: 选择测评

导航到 **发现** 页面，按类别浏览可用的测评：

- 🧠 人格与性格
- 😌 心理健康与幸福
- ❤️ 关系与社交
- 💼 职业与专业发展
- 🎮 娱乐与趣味

### 步骤 2: 完成测评

1. 选择你感兴趣的测评
2. 阅读介绍和说明
3. 诚实地回答所有问题
4. 提交你的回答

### 步骤 3: 查看结果

- 📊 查看详细分析和图表
- 🎯 获取个性化推荐
- 📚 探索相关知识库
- 🔄 与他人比较（匿名）

### 步骤 4: 开始成长之旅

1. 📋 根据结果接收训练推荐
2. 🏋️ 选择你感兴趣的训练项目
3. 📈 追踪你的进度
4. 🎖️ 解锁成就和徽章

