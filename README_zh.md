### 🎯 测评 - 训练 完整闭环

```
做完测评 → 自动识别极端分 → 智能推荐对应训练
    ↓                                            ↑
记录进步数据 ← 完成训练 ← 进入专属成长方案
```

> **举个例子：**
> - 宜人性 A 88分 (讨好者) → 「讨好者边界建立训练」
> - 尽责性 C 92分 (工作狂) → 「完美主义排毒训练」
> - 神经质 N 85分 → 「情绪风暴平息术」
> - 焦虑型依恋 80分 → 「安全感自给自足训练」

### 🚀 快速开始

```bash
# 克隆项目
git clone https://github.com/badhope/MindMirror.git

# 进入目录
cd MindMirror

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 生产构建
npm run build
```

### 📁 项目结构

```
MindMirror/
├── src/
│   ├── app/
│   │   ├── assessments/          # 📊 43个测评计算器
│   │   ├── components/           # 🧩 通用组件
│   │   │   └── training/         # 🏋️ 训练引擎
│   │   ├── data/                 # 📚 数据层
│   │   │   ├── assessments/      # 测评定义
│   │   │   ├── training-cognition.ts     # 思维认知训练
│   │   │   ├── training-emotion.ts        # 情绪管理训练
│   │   │   ├── training-attachment.ts     # 依恋关系训练
│   │   │   ├── training-social-fun.ts     # 社交+趣味训练
│   │   │   └── training-library.ts        # 训练总索引
│   │   ├── pages/                # 📄 页面入口
│   │   └── report/               # 📈 报告模板
│   ├── engine/                   # ⚙️ 计算引擎
│   └── store/                    # 📦 状态管理
```
