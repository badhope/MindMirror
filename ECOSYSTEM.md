## 📦 NPM 包发布系统（零注册，一次操作）

### 自动 CDN 全球曝光！

#### 第一步：发布到 NPM，全球 CDN 自动收录！

```bash
# 1. 初始化 package.json
npm login      # 一次登录，永久使用
npm publish # 一行命令发布！
```

**发布后自动出现在：**
✅ npmjs.com/package/humanos
✅ jsdelivr.com/package/npm/humanos
✅ unpkg.com/humanos
✅ Libraries.io/npm/humanos
✅ Openbase.com/npm/humanos
✅ 全球几十个包管理器自动爬取！
