# 国家模拟引擎后端化 - 详细实施计划

## 📋 项目概述

**当前痛点：**
- 国家模拟tick循环完全阻塞JavaScript主线程
- 快进模式下UI完全无响应
- 计算速度受浏览器单线程限制
- 刷新页面进度丢失

**预期收益：**
- ✅ 计算性能提升 **8-9倍**
- ✅ UI永不卡顿（计算与渲染分离）
- ✅ 支持后台计算+多标签页同步
- ✅ 断点续算（刷新不丢失进度）
- ✅ 多人联机对战基础架构

## 🔄 迁移风险与降级策略

| 风险 | 概率 | 应对措施 |
|------|------|---------|
| 前后端计算不一致 | 中 | 双写验证中间件，抽样1%对比，偏差>0.1%自动切前端 |
| 网络延迟影响体验 | 中 | 本地预渲染+乐观更新，延迟>200ms自动切前端 |
| 后端过载 | 低 | 服务端限流+降级到前端计算 |
| 迁移工作量超预期 | 低 | 渐进式：先经济系统，再人口，最后AI |
